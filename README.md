O código fornecido irá ler um número inteiro n da entrada padrão (STDIN).

Sem usar nenhum método de string, tente imprimir o seguinte:

123...n


Observe que "..." representa os valores consecutivos entre esses números.

Exemplo

Se n = 5

Imprima a string:

12345

Formato de Entrada

A primeira linha contém um inteiro n.

Restrições
1 ≤ n ≤ 150

Formato de Saída

Imprima a lista de números inteiros de 1 até n como uma única string, sem espaços.

Entrada de Exemplo 0
3

Saída de Exemplo 0
123
